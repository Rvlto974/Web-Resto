3. Typographie

Police des titres : Playfair Display

Playfair Display Bold

Playfair Display SemiBold

Playfair Display Regular

Usage : Titres principaux (h1, h2, h3), logo, citations

Caracteristiques : Serif elegant, evoque le savoir-faire traditionnel

Police du corps : Open Sans

Open Sans Bold - Pour les elements importants

Open Sans SemiBold - Pour les sous-titres

Open Sans Regular - Pour le texte courant

Open Sans Light - Pour les textes longs

Usage : Paragraphes, boutons, navigation, formulaires

Caracteristiques : Sans-serif moderne, excellente lisibilite

Echelle typographique

H1 - 2.5rem (40px) Titre Principal
H2 - 2rem (32px) Titre de Section
H3 - 1.5rem (24px) Sous-titre
H4 - 1.25rem (20px) Petit titre
Body - 1rem (16px) Texte courant
Small - 0.875rem (14px) Texte secondaire
Caption - 0.75rem (12px) Legende, mentions

5. Boutons

7. Icones

Bibliotheque : Font Awesome 6

Utiliser les icones Font Awesome en style "solid" pour la coherence.


Icones principales utilisees :

  • Navigation : fa-home, fa-utensils, fa-envelope, fa-user
  • Actions : fa-shopping-cart, fa-plus, fa-edit, fa-trash
  • Statuts : fa-check, fa-times, fa-clock, fa-truck
  • Social : fa-facebook, fa-instagram, fa-twitter
  • Contact : fa-phone, fa-map-marker-alt, fa-envelope
  • Allergenes : fa-wheat-awn, fa-egg, fa-fish, fa-cow

Taille standard : 1rem (16px) pour inline, 1.5rem (24px) pour boutons

Couleur : Heriter de la couleur du texte parent

8. Bonnes Pratiques

A FAIRE
  • Utiliser les couleurs definies dans cette charte
  • Respecter la hierarchie typographique
  • Maintenir des contrastes suffisants (RGAA)
  • Utiliser des images de haute qualite pour les menus
  • Garder une coherence visuelle sur toutes les pages
A EVITER
  • Ajouter des couleurs non definies dans la palette
  • Utiliser plus de 2 polices differentes
  • Deformer ou modifier le logo
  • Surcharger les pages avec trop d'elements
  • Utiliser des images de mauvaise qualite

9. Design Responsive

Points de rupture (Breakpoints)

Nom Largeur Appareils
Mobile < 576px Smartphones portrait
Tablet 576px - 991px Smartphones paysage, tablettes
Desktop 992px - 1199px Ordinateurs portables
Large >= 1200px Grands ecrans

Adaptations mobile

  • Navigation : Menu hamburger avec drawer lateral
  • Cartes : Affichage en colonne unique
  • Typographie : Reduire les tailles de 10-15%
  • Boutons : Pleine largeur sur mobile
  • Filtres : Drawer/modal au lieu de sidebar
